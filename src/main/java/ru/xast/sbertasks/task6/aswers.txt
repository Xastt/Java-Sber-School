
1) Аннотацией @Before помечаются методы, которые буду выполняться до выполнения
каждого тестового метода. Помогает убрать лишний повторяющийся код.

2)Для того, чтобы проверить выбросил ли метод исключение есть метод
assertThrows(само исключение, код где исключение может быть выброшено)

3)Mock - создает полностью виртуальный объект.
  Spy - оборачивает уже существующий объект в обертку.

4)Используется для того, чтобы убедиться, что тестируемый класс вызвал нужные
методы нужных объектов Mockito.verify(объект).имяМетода(параметр). Но иногда
нужно проверить не только вызов но и кол-во вызываемых раз
для этого используется: Mockito.verify(объект,количество).имяМетода(параметр);
